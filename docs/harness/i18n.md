# i18n 규칙

## 지원 언어

- 기본 언어: `ko`
- 지원 언어: `ko`, `en`

## 규칙

- 사용자가 보는 문구가 공용 문구이거나 페이지 단위 문구라면 `src/i18n/dictionaries.ts`에서 가져온다.
- 한국어 문구를 기준 원본으로 삼는다.
- 같은 key에 대해 영어 문구도 함께 유지한다.
- 프로젝트 데이터의 다국어 필드는 `src/i18n/dictionaries.ts`에 둘 수 있다.
- 임시 문구가 아니라면 페이지 컴포넌트에 제목, 라벨, 설명 문구를 하드코딩하지 않는다.

## 현재 범위

현재 프로젝트는 정적 기본 언어 헬퍼를 사용한다. 언어별 라우팅은 나중에 다음 형태로 추가할 수 있다.

```text
/ko
/en
```

언어별 라우팅을 추가하기 전까지는 한국어 사전을 기본으로 렌더링한다.
